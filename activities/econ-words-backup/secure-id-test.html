Secure ID Generation Test

This page tests the cryptographically secure guest ID generation implemented in the EconWords authentication service.

It verifies:

  • The use of the Web Crypto API instead of Math.random()
  • The absence of collisions in a large sample of generated IDs
  • The entropy and randomness quality of the generated IDs

How It Works

The secure ID generation uses the Web Crypto API which provides cryptographically strong random values:

function generateSecureId(length = 16) {
    // Create a typed array to hold the random values
    const randomValues = new Uint8Array(length);
    
    // Fill the array with cryptographically secure random values
    crypto.getRandomValues(randomValues);
    
    // Convert to a string with letters and numbers
    return Array.from(randomValues)
        .map(val => (val % 36).toString(36))
        .join('');
}
